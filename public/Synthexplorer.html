Modulation

La modulation utilise un signal de contrôle pour faire varier les paramètres d'un autre signal. Le LFO (Low Frequency Oscillator) génère des signaux à basse fréquence (0.1-20Hz) pour moduler d'autres paramètres.

Types de modulation :

  • Vibrato : Modulation de fréquence (FM) - variation de hauteur
  • Trémolo : Modulation d'amplitude (AM) - variation de volume
  • Modulation de filtre : Variation du cutoff pour effets de balayage

Filtres

Les filtres sculptent le spectre fréquentiel d'un son. Essentiels en synthèse soustractive pour créer des timbres variés à partir d'ondes riches harmoniquement.

Types de filtres :

  • Passe-bas : Laisse passer les basses, coupe les aigus
  • Passe-haut : Laisse passer les aigus, coupe les basses
  • Passe-bande : Laisse passer une bande de fréquences
  • Coupe-bande : Coupe une bande spécifique

Paramètres clés :

  • Fréquence de coupure : Point où le filtre commence à agir
  • Résonance (Q) : Amplification autour de la fréquence de coupure
  • Pente : Rapidité de coupure (6dB, 12dB, 24dB par octave)

Enveloppes ADSR

L'enveloppe ADSR contrôle l'évolution temporelle de l'amplitude d'un son, imitant le comportement naturel des instruments acoustiques.

Phases ADSR :

  • Attack : Temps pour atteindre le niveau maximum
  • Decay : Temps pour descendre au niveau de sustain
  • Sustain : Niveau maintenu tant que la note est pressée
  • Release : Temps pour revenir à zéro après relâchement

Applications :

  • VCA : Contrôle d'amplitude
  • VCF : Modulation du filtre
  • VCO : Modulation de pitch pour effets d'attaque

Synthèse Soustractive

La synthèse soustractive part d'un signal riche harmoniquement et utilise des filtres pour "soustraire" des fréquences et sculpter le timbre.

Chaîne de traitement typique :

VCO → Mixer → VCF → VCA → Sortie
Avec enveloppes et LFO pour moduler les paramètres

Composants principaux :

  • VCO (Voltage Controlled Oscillator) : Génère les formes d'ondes de base
  • Mixer : Mélange plusieurs oscillateurs
  • VCF (Voltage Controlled Filter) : Filtre les fréquences
  • VCA (Voltage Controlled Amplifier) : Contrôle l'amplitude finale
  • Enveloppes : Modulent les paramètres dans le temps

Synthèse FM

La synthèse FM utilise un oscillateur (modulateur) pour faire varier la fréquence d'un autre (porteur), créant des harmoniques complexes.

Principe mathématique :

Sortie = sin(2π × fc × t + I × sin(2π × fm × t))

Où :

  • fc : Fréquence porteuse
  • fm : Fréquence modulatrice
  • I : Index de modulation (profondeur)

Caractéristiques :

  • Ratio entier : Harmoniques musicales (1:1, 2:1, 3:2...)
  • Ratio non-entier : Timbres inharmoniques, métalliques
  • Index élevé : Plus d'harmoniques, son plus complexe

Synthèse Additive

La synthèse additive construit des sons complexes en additionnant plusieurs ondes sinusoïdales (harmoniques). Contrôle précis du spectre harmonique.

Principe :

Son Final = Σ(Amplitude_n × sin(2π × n × f × t + Phase_n))

Avantages :

  • Contrôle précis : de chaque harmonique
  • Timbres organiques : reproductibles
  • Évolution temporelle : par harmonique
  • Sons impossibles : avec spectres non-naturels

Wavetables

La synthèse par wavetables utilise des échantillons pré-calculés de formes d'ondes complexes. Morphing entre différentes tables pour évolution timbrale.

Avantages :

  • Qualité audio : Formes d'ondes complexes précalculées
  • Morphing : Transition fluide entre formes
  • Efficacité CPU : Moins de calculs temps réel
  • Contrôle temporel : Évolution du timbre dans le temps

Technique :

  • Taille : 256-1024 échantillons par cycle
  • Interpolation : Linéaire/cubique entre échantillons
  • Anti-aliasing : Tables multiples selon fréquence
  • Morphing : Interpolation entre tables adjacentes

Synthétiseurs Classiques

Les synthétiseurs analogiques classiques ont défini le son de la musique électronique.

Moog Minimoog Model D (1970) :

VCO 1 → Mixer → Ladder Filter → VCA → Output
VCO 2 → ↑
VCO 3 → ↑
Noise → ↑

  • Filtre Ladder 4 pôles (-24dB/octave), saturation douce
  • 3 VCO avec réglage fin, sync, modulation FM
  • Circuit analogique à composants discrets
  • Enveloppe ADS (pas de Release séparé)

ARP Odyssey (1972) :

  • Filtre variable 2/4 pôles commutable
  • Sample & Hold pour séquences aléatoires
  • Ring Modulator pour sons métalliques
  • Proportional Pitch Control

Sequential Prophet-5 (1978) :

  • Premier polyphonique programmable (5 voix)
  • Mémoire de patches (40 presets)
  • 2 VCO par voix pour sons épais
  • Curtis chips VCO et VCF intégrés

Roland TB-303

Le Roland TB-303, conçu pour accompagner les guitaristes, est devenu l'icône de l'acid house. Son filtre résonnant et ses glissements créent le son acid unique.

Caractéristiques techniques :

  • VCO : Dents de scie uniquement, glissements (slides)
  • Filtre : Passe-bas 18dB/octave avec résonance élevée
  • Enveloppe : Générateur simple pour filter et amplitude
  • Accent : Accentuation dynamique des notes
  • Séquenceur : 16 pas avec slides et accents

Chaîne signal :

VCO Saw → 18dB Filter → VCA → Distortion
Env Gen → Filter Cutoff
Accent → Env Amount

Synthèse Granulaire

Décompose le son en microscopiques "grains" de 1-100ms manipulables individuellement. Crée des textures impossibles avec méthodes traditionnelles.

Paramètres :

  • Taille grain : Durée de chaque micro-échantillon
  • Densité : Nombre de grains par seconde
  • Dispersion temporelle : Variation aléatoire du timing
  • Dispersion pitch : Variation de hauteur entre grains
