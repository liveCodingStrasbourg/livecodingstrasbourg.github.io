LIVE MARKET CODING

Documentation en français

Introduction

LIVE MARKET CODING est un environnement de génération sonore basé sur le thème d'un supermarché capitaliste. Utilisez des commandes textuelles pour créer une composition sonore en direct, en manipulant votre caddie et en ajoutant des produits avec différents attributs. Cette expérience sonore constitue une critique ludique du consumérisme et du capitalisme, transformant l'acte d'achat en création musicale.

⚠️ ATTENTION : Ceci est une version alpha. Des bugs peuvent survenir, notamment dans les modes spéciaux.

Commandes de base

Manipulation du caddie

Le caddie représente la section rythmique de votre composition. Différents types de roues produisent différents rythmes.

my cart has square wheels    // Rythme basique, carré
my cart has broken wheels   // Rythme irrégulier
my cart has premium wheels  // Rythme précis et constant
my cart has defective wheels // Rythme glitchy
my cart has bargain wheels  // Rapide mais inconstant
my cart has luxury wheels   // Précis avec variations subtiles
my cart has no wheels       // Silence

Ajouter des produits

Les produits sont des synths qui créent la mélodie et les textures sonores. Chaque produit a son propre son distinctif.

add beer                    // Ajoute une bière (synth FM avec harmoniques)
add salad                   // Ajoute une salade (synth AM à onde carrée)
add ham                     // Ajoute du jambon (synth Mono à onde en dents de scie)
add milk                    // Ajoute du lait (synth à corde pincée)
add chips                   // Ajoute des chips (synth de bruit blanc)
// etc.

Modes spéciaux

Les modes affectent la sonorité globale et l'ambiance.

discount mode on            // Mode réduction - détune aléatoire
inflation mode on           // Mode inflation - hauteur qui augmente progressivement
consumerism mode on         // Mode consumérisme - surtention des produits
black_friday mode on        // Mode Black Friday - chaos total
aisle_7 ambience on         // Mode allée 7 - réverbération inquiétante
fluorescent_lights flicker on // Mode clignotement - tremolo subtil

Tous les modes peuvent être désactivés en remplaçant on par off.

Modificateurs de produits

Les modificateurs changent les caractéristiques sonores des produits. Ajoutez-les avant le nom du produit.

Modificateurs de hauteur

Modificateur Effet
fresh Augmente la hauteur d'une octave
old Diminue la hauteur d'une octave

Modificateurs de filtre

Modificateur Effet
strong Filtre passe-bas, accentue les basses
flavorless Filtre passe-haut, accentue les aigus

Modificateurs d'effets

Modificateur Effet
cheap Effet bitcrusher, réduction de la qualité
expensive Reverb, son plus ample
processed Chorus, son plus épais
industrial Distortion, son sale
overpriced Phaser, balayage du son
vomit Chebyshev, distortion extrême
artisanal Tremolo, volume ondulant
bargain Delay avec feedback modéré
luxury Reverb importante
artificial Vibrato, hauteur ondulante
mass-produced Bitcrusher intense
addictive Ping-pong delay, répétitions stéréo

Exemples combinés

add fresh strong beer        // Bière fraîche et forte
add old flavorless expensive milk // Lait périmé sans goût et cher
add cheap processed industrial ham // Jambon bas de gamme industriel

Paramètres spéciaux

Ces paramètres ajoutent des caractéristiques spécifiques aux produits.

Nutriscore

Modifie la tonalité du produit.

add beer nutriscore A       // Bière en tonalité A
add ham nutriscore C        // Jambon en tonalité C

Durée de conservation

Définit la fréquence de répétition du produit.

add milk shelflife today    // Répétition très rapide
add wine shelflife decade   // Répétition très lente

Options : today, week, month, year, decade, forever

État du produit

add beer open              // Produit ouvert, comportement imprévisible

Combinaisons complexes

add old beer nutriscore C shelflife month open

Raccourcis clavier

Raccourci Action
Ctrl + Enter Exécuter la ligne courante
Ctrl + Shift + Enter Exécuter toutes les lignes
Ctrl + R Générer une commande aléatoire
Escape Tout arrêter

Conseils pratiques

  • Commencez par définir les roues de votre caddie pour établir un rythme de base.
  • Ajoutez des produits un par un pour comprendre leur son spécifique.
  • Expérimentez avec différents modificateurs pour créer des textures sonores intéressantes.
  • Les modes spéciaux peuvent créer des effets puissants mais sont aussi plus instables.
  • Utilisez la commande de génération aléatoire pour découvrir de nouvelles combinaisons.
  • Si le son devient trop chaotique, utilisez le bouton "Tout Arrêter" et recommencez.
